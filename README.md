## Usage

1. **Clone the repository:**
    ```sh
    git clone https://github.com/yourusername/test-frontend-engineer.git
    cd test-frontend-engineer
    ```

2. **Install dependencies:**
    ```sh
    npm install
    ```

3. **Run the development server:**
    ```sh
    npm start
    ```

4. **Run tests:**
    ```sh
    npm test
    ```

5. **Build for production:**
    ```sh
    npm run build
    ```
